Features

GET
  • Return list of all movies to user
  • Return movie data about a single movie
  • Return data about a movie's genre
  • Return data about a director by name
POST
  • Allow a user to register
PUT
  • Allow user to update their user name
  • Allow user to add a movie from the movie list to their favourites.
DELETE
  • Allow users to remove a movie from their favourites
  • Allow existing user to deregister

HTML methods and specifications

  • GET /movies Returns a list of movies. Data format: JSON
  • GET /movies/:title Returns a movie by its title. Data format: JSON.
  • GET /movies/genre/:genre Returns details of a genre by name. Data format: JSON.
  • GET /movies/director/:name Returns a director's details by their name. Data format: JSON.
  • POST /users Creates a user. Data format: JSON. Expected Body: { "name": "name" "userMovies": [] }
  • PUT /users/:userName/:newName Updates a user's username. Data format: JSON.
  • PUT /users/:userName/addMovie/:title Adds a movie to the user's favourites. Data format: JSON.
  • DELETE /users/:userName/deleteMovie/:title Removes a movie from the user's favourites.
  • DELETE /users/:id Removes a user
